## Mini Guia

### Variáveis

Servem para armazenar informações.

Exemplo

```python
nome = "Bruno"
idade = 15
```

### If

Serve para tomar decisões.

```python
if idade >= 18:
    print("Maior de idade")
```

### For

Utilizado para repetir ações.

```python
for i in range(5):
    print(i)
```
